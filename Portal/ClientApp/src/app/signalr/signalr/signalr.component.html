ABOUT

Adds real-time functionality to web applications.

Server side-code pushes content to clients instantly.

Communication is handled by WebSockets, Server-Sent Events or Long Polling depending on capabilities of the server and client.

Startup.cs: Register SignalR middleware

NPM: @aspnet/signalr
